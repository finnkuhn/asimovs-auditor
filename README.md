# Asimovs-Auditor

This is the official bot for moderating r/selfhosted. 
When a new Post is made it comments under the post and asks how AI was used. 
When a user comments under the comment, it gets locked and the post approved.

Also verfies if a project is < 3 months by checking it's github repository.


Based on the tja-tja bot made by u/Sajeg and adopted for r/selfhosted by u/FnnKnn
